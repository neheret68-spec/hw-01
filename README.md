# English Excellence 2.0 — Лабораторна робота 1

HTML-розмітка першої сторінки лендингу **English Excellence 2.0**.

## Структура проекту

- `index.html` — головний HTML-файл;
- `images/` — папка із зображеннями для сторінки.

## Що виконано

- використано семантичні теги `header`, `main`, `section`, `article`, `footer`;
- логотип зроблено посиланням на головну сторінку;
- усі зображення мають атрибути `src`, `width`, `height` та `alt`;
- CSS не використовується, тому що в лабораторній роботі потрібно виконати тільки HTML-розмітку.
- Посилання https://neheret68-spec.github.io/hw-01/
